# Media folder

Replace the placeholder files with your own:

- `image1.jpg` … `image6.jpg`  — 6 gallery photos of Santoshi
- `video1.mp4` … `video4.mp4`  — 4 short video clips
- `loved1.jpg` … `loved5.jpg`  — 5 photos of loved ones (square works best)
- `music.mp3`                  — background music (a soft custom track is included; replace with your own anytime)

## Change the music
Just overwrite `music.mp3` with any `.mp3` file (keep the same filename), or edit
`birthday/templates/birthday/index.html` to point the `<audio id="bgMusic">` tag
at a different file.
